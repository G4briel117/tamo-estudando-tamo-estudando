Trhee Tier - Três camadas
- Ou seja temos essa aplicação dividida em três partes 

- A Primeira camada -> Chama uma API, via rede
- A segunda camada - Armazena o dado
- A Terceira camada tem o dado armazenado 
- A terceira camada passa a resposta para o back que volta para o front e retorna para o requisitante
