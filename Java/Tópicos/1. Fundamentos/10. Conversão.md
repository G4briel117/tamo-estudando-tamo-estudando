### Conversão dos numeros primitivos

```Java
double a = 1;

System.out.println(a);

float b = (float) 1.12345678;

//Esse é uma conversão explicita, pois você esta indicando qual tipo você quer que essa classe seja

System.out.println(b);

int c = 4;

byte d =(byte)c;// Percebe-se que se você colocar uma numero que não é suportado pelo byte que é no maximo 127, ele vai partir para o proximo da fila e recomeçar a contagem, ou seja, se voce colocasse 128, ele ja iria pra algo como -126 ou -127 na conversão la no console.

System.out.println(d);

```

lembrando que esses são para [[Tipos Primitivos]]

## Conversão de Número -> String 


```Java

Integer numb = 10000;

System.out.println(numb.toString().length());

//Converteu os numeros em string do tipo inteiro para string

int num2 = 100000;

System.out.println(Integer.toString(num2));

//Outra forma de converter os numeros

```


## Conversão de String -> Número


```Java

double numb1 = Double.parseDouble(variavel que é String);
```

Do mesmo jeito que existe o `Integer.parseInteger();` existe o `Double.parseDouble();`


```Java
//o Double.paserDouble(String); é usado para converter a String em tipo Double.

String value1 = "2";
String value2 = "3";

sysout(value1+value2);

Double numb1 = Double.parseDouble(value1);
Double numb2 = Double.parseDouble(value2);

Double result = numb1 + numb2;

sysout(result);

// Aqui um exemplo de conversão de String para Numero. A mesma coisa se aplica para o conjunto "Integer"




```


O mesmo conceito seria possivel com o `Integer` e outros tipos.
