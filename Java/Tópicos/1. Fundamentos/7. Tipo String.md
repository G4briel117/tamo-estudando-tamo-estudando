## Concatenação de String

Para concatenar, isso é "juntar ou adicionar" uma string a outra string, a gente usa `.concat("")`

```Java
String s = "Opa iai vei belezi ?";

System.out.println(s.concat("!!!!"));
// Vai retornar ""Opa iai vei belezi ? !!!!"

System.out.println(s +"@@@@");
/// Vai retornar "Opa iai vei belezi ?@@@@"
```

Como a gente em ambos os casos usando a *function* como o `+` dá o mesmo resultado.

## String iniciada e terminada com determinada "palavra ou letra"

Para saber se a String inicia com certa palavra. utilizamos uma `Função` Ela é booleana

```Java

String s = "Opa iai vei belezi ?";


System.out.println(s.startsWith("Opa"));

//true

System.out.println(s.startsWith("opa"));

//false

```

perceba que se colocar *letra minuscula* ela dá como falso.

*porém* se a gente converter a string para *loweCase* ela entende como `true`

```Java
String s = "Opa iai vei belezi ?";

System.out.println(s.toLowerCase().startsWith("opa"));
//true
```

**Atenção nisso**

E para saber se ela termina com determinada palavra utilizamos ``

## PrintF

Para não ter um monte de coisas de concatenação de strings que é o `+` nas strings, iremos usar o `format`

```Java
System.out.printf("O senhor %s %s tem %d anos e ganha R$%.2f",nome, sobrenome, idade, salario);

//%s nome, %s sobrenome, %d idade, %fsalario.

// %s representa String, %d numeros inteiros, %f numeros com pontos flutuantes e colocando por exemplo %2f voce limita a apenas duas casas decimais
```
