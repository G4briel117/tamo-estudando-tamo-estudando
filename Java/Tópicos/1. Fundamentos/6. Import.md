## Biblioteca Scanner

```Java
import java.util.Scanner; //<- esse é o import do Scanner

Scanner leia = new Scanner(System.in);
//<- essa é a ativação do scanner para o uso. Na palavra "leia" você pode usar qualquer palavra, pois é ela que usará essas coisas
```

`Lembrando que a saida padrão o ".out" é a tela e a entrada é o teclado que é o ".in" como no caso do Scanner `

### Entradas no Scanner


```Java
leia.nextLine(); - "String"
leia.nextDouble(); - "numeros Double"
leia.nextInt(); - numeros inteiros

```
