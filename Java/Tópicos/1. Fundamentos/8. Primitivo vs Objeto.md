
O Java tem 8 tipos de primitivos.

Sendo eles com:
- 2 bits
- 4 bits
- 8 bits
- [[Tipos Primitivos|para saber mais sobre. CLIQUE AQUI]]


`A partir do momento que você cria uma classe e associa um valor a essa classe, isso significa que aquele valor associado é um objeto.`

##### Ex:
```Java
String classe = "objeto"
```

`Esse "Objeto" tem atributos e comportamentos`

#### Ex:
```Java

classe.toUpperCase();
//OBJETO
//ou seja, um corportamento com o valor da classe (Objeto) foi criado
```

A grande diferença para os [[Tipos Primitivos]] para os Objetos é que:

Objeto: Tem [[5. Notação Ponto|Notação Ponto]]
Primitivos: Não tem Notação Ponto

`Objetos só guardam valor`

`Tudo em Java são Objetos, menos tipos Primitivos`

`wrappers são versão objetos dos tipos primitivos`
