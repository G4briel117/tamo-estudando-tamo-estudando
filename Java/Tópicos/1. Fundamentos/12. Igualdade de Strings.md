
```Java

System.out.println("2" == "2"); //true

String s = new String("2");
System.out.println("2"==s);//false
```
Essa é uma das maneiras de se usar igualdade de String, porem muitas vezes ela da um falso retorno.

```Java
String s = new String("2");
System.out.println("2".equal(s))// true
```
Essa forma esta correta, pq ele vai comparar os conteudos.
```Java
//outro exemplo é assim

Scanner ler = new Scanner(System.in);

String b2 = ler.nextLine().trim();
System.out.println("2".equals(b2.trim()));//true
```
esse `.trim()` tira os espaços na hora da respostas

#### O ideal é que a gente use .equals() para comparação de strings.
