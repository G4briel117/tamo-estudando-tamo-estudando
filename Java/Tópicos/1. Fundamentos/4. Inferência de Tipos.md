A palavra reservada `var` identifica o tipo primitivo que irá ser usada, ou seja, ela identifica os [[Tipos Primitivos]]. Após ter definido o tipo de primitivo não é possível muda-lo em outra variável.

![[Pasted image 20250106104256.png]]

Com o `var` é possível definir uma variável """""depois""""", é necessário definir ela na hora.

```Java
var c;
c = 1087.56;
```

*INCORRETO*
Isso não funciona, pois é necessário inicializar a variável com o mesmo valor na mesma linha, pois o Java não consegue definir o tipo da variável em outra linha.

``` Java
var c = 1087.56;
```

*CORRETO*
Desse jeito o Java consegue interpretar e definir o tipo da variável.
