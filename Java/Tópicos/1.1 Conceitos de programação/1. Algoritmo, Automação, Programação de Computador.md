
### Algoritmo

Sequência finita de instruções para resolver um problema.

##### Exemplo:
###### Problema:
- Lavar roupa suja

###### Algoritmo
- Colocar a roupa no recipiente
- Colar um pouco de sabão
- Encher de água
- Mexer tudo até dissolver todo o sabão
- Deixar de molho por vinte minutos
- Esfregar a roupa
- Enxaguar
- Torcer

### Automação

Consiste em utilizar maquina(s) para executar o procedimento desejado de forma automática ou semiautomática.

##### Exemplo

Na parte de lavar roupa suja poderíamos substituir por uma maquina que lava roupa automaticamente. A partir da etapa 3 

O que ambos tem a ver com programação de computadores ?

Programas de computador são algoritmos executados pelo computador (em linhas gerais)
