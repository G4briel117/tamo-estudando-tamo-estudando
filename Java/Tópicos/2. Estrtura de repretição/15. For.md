```Java
//dividimos a nossa sentença "for" em tres partes 
for(a; b; c){
}


```

Dividindo em três partes o código que foi dito e nesse caso seria algo assim:

`a = varivael int i = 0`
`b = expressão (sysout por exemplo)`
`c = incremento i++ `

Essa estrutura, foi planejado e determinada para você ter um numero limitado de repetições.

```Java
int i = 0

for (int i; i<10; i++ ){
System.out.printf("i = %d", i);
}


```


### For dentro do outro

Para entender o laço `for` dentro de um `for` a gente precisa pensar que ele primeiro vai rodar o código de dentro e depois o de fora

````Java

for (int i = 0;i<10;i++){
	for(int j = 0; j<10; j++){
	
	System.out.printf("[%d %d]",i, j)
	
	}

}
````
