## 13.1 Else

Ele `Else` serve para quando não é atendido a condição imposta, então ela fará o trabalho.

```Java
double numb1 = 10;
double numb2 = 10;
double result = numb1 % numb2;

if (result == 0) { // pode ser feito sem estar em bloco de programação, porem apenas o primeiro texto será feito.
System.out.println("Par");
}Else{
	System.out.println("Impar");
}//se não for par, será impar


```
