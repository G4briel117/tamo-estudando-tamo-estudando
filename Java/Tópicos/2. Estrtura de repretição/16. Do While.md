
Do while tem como estrutura 


```Java

Do { //sentença

}while(//condição) ;// Isso mostra que independente de ser verdadeiro ou falso o bloco de código vai se repetir pelo menos uma vez.


```


#### Example !

```Java

package controle;
import java.util.Scanner;

public class Dowhile {

public static void main(String[] args) {

Scanner ler = new Scanner(System.in);

  

String text = "";

do {

System.out.println("Voce comeu o pão ? Se você comeu digite Sou omisso");

text = ler.nextLine();

}while(!text.equalsIgnoreCase("Sou omisso"));

System.out.println("Sabia que você tinha comido o pão !!!! Seu omissooooo");

}

}

```
