
Usado para repetir uma sentença de código ou o bloco inteiro !
*While = Enquanto*

Se eu quisesse uma determinada sentança se repetisse 10 vezes


```Java

int i = 1 

while(i<10){// dentro do while eu vou ter uma expressão que vai retornar "true" ou "false" e se for true ela irá se repetir até se tornar falsa
System.out.println("i = %d ", i);//Minha sentença que vai se repetir e mostrar os numero crescendo 

i++ //incremento onde ira contabilizar i+1 até ele se repetir e ser maior qiue o 10 ou o numero colocado lá

}


```
