### O que é uma API Drift ?

###### R: Inconsistência de no código que passam de forma despercebidas.
- A diferença entre o comportamento que foi documentado e o comportamento real da inerface.
- Podemos atribuir esses "Drifit" a mudanças mal feitas, implantações rápidas, revisões sem critérios... Inúmeros problemas podem acontecer quando se fala de API Behavior Drift
### Possíveis Problemas enfrentado:
- Ocorre quando há diferença das especificações documentadas
- Mais trabalho em excesso para a solução do problema
- Exposição de dados para o banco de dados da interface
- 
**Ex: Atributo que deve retorna type: int acaba retornando String. 
- Quebra do consumo da API
- Retrabalho no API 
- Evolução desnecessária para a solução do trabalho
